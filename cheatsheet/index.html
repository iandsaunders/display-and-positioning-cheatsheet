Display and Positioning

Z-Index

Attribute Value Use
z-index Low and High Range Specifies how far back or how forward an element will appear on a web page

Position

Attribute Value Use
Position Fixed Fixes element no matter where you scroll. Ex: Navigation Bar
Position Static Always positioned according to the normal flow of the page
Position Relative An element positioned with regard to top, left, right, bottom rules such as margin and border
Position Absolute Positioned relative to the nearest positioned ancestor. Removed from normal flow and can overlap elements
Position Sticky A fixed element that can 'stick' to certain areas and stop scrolling with the user

Display

Attribute Value Use
Display Block Take up full width of the container with line breaks before and after
Display Inline Take up as little space as possible, flow horizontally, and cannot have their height or width adjusted
Display Inline-Block Can appear next to each other, and can have their height and width manually adjusted
